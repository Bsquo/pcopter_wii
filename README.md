# MiniCopter: Adventure Flight

Also known as *Radio Helicopter* in Europe, and *プチコプターWii アドベンチャーフライト* (*Puchi Copter Wii: Adventure Flight*) in Japan.

Currently, only the European version is supported. It builds the following DOL:

pcopter_wii_eur.dol: `sha1: 772bec0b25fa6f947701b3355823608ad8171a27`

## Build Instructions

1. Obtain a copy of mwcceppc and mwldeppc from CodeWarrior (GC 3.0a3). Also obtain powerpc-eabi-as and powerpc-eabi-objcopy, which can be installed with [devkitPro](https://devkitpro.org/wiki/Getting_Started). These will go into the `tools` folder.
3. Run `make`.
